# Chip-workshop

Портфолио-визитка: одностраничный сайт, собранный в Chip-workshop. Тёмная тема в стиле BMW Isle of Man Green, анимация частиц (реакция на курсор и клики). На сайте есть чат-бот: отвечает на вопросы об авторе, услугах, контактах и может поделиться фактом о типографике и минимализме (цитата Дитера Рамса).

## Техническая информация

- **Стек:** статический HTML/CSS/JS, один файл `index.html` (вёрстка, стили, скрипт частиц, чат-бот).
- **Файлы:** `index.html`, `avatar.jpg`; опционально `vercel.json` для деплоя на Vercel.
- **Локальный запуск:** открой `index.html` в браузере или `npx serve .` / `python -m http.server 8080`.
- **Лицензия:** MIT — см. [LICENSE](LICENSE).

## Ветки и слияние
